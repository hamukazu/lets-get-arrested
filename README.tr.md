# Hadi-Tutuklan Projesi

Japonya'da ilkokula giden bir kız öğrenci bir web sayfasına şu şekilde sonsuz bir JavaScript döngüsü içeren bir sayfanın bağlantısını koyduğu için polisler tarafından tutuklandı:
```js
while (1) {
  alert("!");
}
```
