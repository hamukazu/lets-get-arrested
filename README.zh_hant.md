```js

  alert("!");
}
```

## 被捕方法

非常簡單，只要 fork 本項目，並建立一個新的 gh-pages branch。結束。當然如果分享 "http://你的帳戶.github.io/lets-get-arrested" 這個 URL 到各大 SNS 平台上，效果更佳。
