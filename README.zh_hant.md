# 大家一起被捕吧計劃

在日本，因為貼了一個前往包含下述無限警告循環的 JavaScript 的網站的鏈接，一名女中學生被警察逮捕並進行輔導：

```js
while (1) {
  alert("!");
}
```

相關新聞：
[https://www3.nhk.or.jp/lnews/kobe/20190304/2020003239.html](https://www3.nhk.or.jp/lnews/kobe/20190304/2020003239.html)

因此這在日本被視為“犯罪”。那麼我們也一起成為罪犯被捕吧！

祝大家犯罪愉快！
